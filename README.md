# viper_mechanicactions
Mechanic Menu


Add items from Inventory_Items folder in your ox_inventory/data/items.lua

Dependencies:
- OX_Inventory
- OX MySQL
- OX_LIB
- Target

To add new jobs go to client/client.lua/line 16 and add more jobs with the lowest grade in job
